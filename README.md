# 3650 Capital

Static multi-page marketing site for 3650 Capital, deployed with GitHub Pages.

## Site Structure

- `index.html`
- `firm.html`
- `solutions.html`
- `transactions.html`
- `leadership.html`
- `contact.html`
- `privacy.html`
- `terms.html`
- `styles.css`
- `motion.js`
- `leadership-data.js`

## Deployment

This repository deploys through GitHub Actions using [deploy-pages.yml](/Users/andrewshea/Desktop/Websites/3650/3650%20Ideas/.github/workflows/deploy-pages.yml).

The published site is:

- `https://heyitsmrshea.github.io/3650-capital/`

## Notes

- Fonts, images, and favicon are stored locally in `assets/`.
- Relative paths are used throughout, so the site works correctly from a GitHub Pages project subpath.
- Vimeo, maps, press links, and selected profile links remain intentional outbound links.
